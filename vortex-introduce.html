✓
Response authenticity — impossible to forge /v1/nodes without the controller's private key.
✓
Code integrity — controller files hashed and cross-checked at every startup. Tampering returns 503 before any traffic is served.
✓
Node identity — nodes must sign registration with their own key. An attacker cannot register on behalf of another node's pubkey.
✓
DNS hijack protection — .sol records live on Solana. An adversary needs to compromise the wallet, not just a DNS registrar.
✓
Handoff security — session transfers carry Ed25519-signed tokens with anti-replay, valid only 5 minutes.
✗
Transport privacy — connection URL visible to your ISP / Cloudflare. Use the Tor mirror if metadata matters.
✗
Message content — NOT touched by the controller. End-to-end encrypted between users. Controller can read nothing if compromised.
✗
Host OS / kernel — if the VPS is rooted, keys can be stolen. Use hardware-backed signing for paranoid deployments.
ACCESS

Always reachable.
Always yours.

Multiple entry points mean no single entity can block your access. Mirror sites serve the same signed content from independent infrastructure.

TUNNEL
Cloudflare tunnel
Free *.trycloudflare.com address. No account required. URL changes when the tunnel restarts — treat as ephemeral.
TOR
Hidden service
.onion address, reachable only through Tor. Censor-proof and metadata-private, but slower. Needs a Tor client.
IPFS
Distributed content
Content-addressed copy of the static frontend. Any IPFS gateway can serve it — no single server to block.
DIRECT
Plain HTTPS
Regular domain + TLS. Fastest option. Blockable at DNS/IP level — fall back to others if censored.
MIRRORS

Alternative hosts serving the exact same signed content as this controller. Each mirror is periodically probed — a green dot means reachable within the last 5 minutes. Tor .onion mirrors can't be health-checked without a local Tor proxy.

GET STARTED

Join the network
in five steps

  1. Install Vortex on your device — mobile or desktop.
  2. Pick Global network mode in the setup wizard.
  3. Paste the controller URL (auto-resolved via SNS) or any entry URL from the Entries page.
  4. Verify the pubkey in your app matches the controller fingerprint below. A mismatch means tampering.
  5. You're connected. Node list, signatures, and routing are handled automatically.
